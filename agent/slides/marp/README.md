# marp/

Marp スライド生成のアセット・テンプレート・テーマを格納する。

## ディレクトリ構成

| ディレクトリ / ファイル | 用途 |
|------------------------|------|
| `project-slide-theme.pptx` | マスタースライドテーマ（PPTX 変換時のベース） |
| `assets/` | ロゴ等の画像アセット |
| `templates/` | スライドテンプレート（Markdown） |
| `themes/` | Marp CSS テーマ |
